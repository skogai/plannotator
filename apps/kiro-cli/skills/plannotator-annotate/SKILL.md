# Plannotator Annotate (Kiro)

Run:

```bash
PLANNOTATOR_ORIGIN=kiro-cli plannotator annotate $ARGUMENTS
```

`$ARGUMENTS` should be a markdown file path, folder path, html file path, or URL.
