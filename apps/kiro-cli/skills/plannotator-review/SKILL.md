# Plannotator Review (Kiro)

Run:

```bash
PLANNOTATOR_ORIGIN=kiro-cli plannotator review
```

You may append an optional PR URL:

```bash
PLANNOTATOR_ORIGIN=kiro-cli plannotator review <pr-url>
```
